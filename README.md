# Wio Pay Sales Calculator

Internal sales-pricing calculator for Wio Pay. Vanilla JS, no build step.

## Scope (v0.1.0 init)

Project skeleton only. Engine, UI and docs land in later commits.

## Run locally

```powershell
npx serve .
# or
python -m http.server 8080
```

Open `http://localhost:3000` (serve) or `http://localhost:8080` (python).
